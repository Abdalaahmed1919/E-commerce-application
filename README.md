## 📌 Overview
**ShopNest** is a full-stack, scalable ASP.NET Core MVC e-commerce application designed with a focus on clean architecture, high-end minimalist UI, and a seamless user experience. 

It provides a complete shopping journey for customers, featuring real-time AJAX filtering and secure cart management, alongside a robust Role-Based Admin Dashboard for managing inventory, categories, and orders.

> **💡 Architecture Highlight:** Built utilizing the **Repository Pattern** to ensure a decoupled, maintainable, and highly testable codebase.

## 🚀 Key Features

### 👤 Customer Experience
* **High-End UI/UX:** A modern, minimalist, and fully responsive design built with Bootstrap 5.
* **Smart Browsing:** Dynamic product filtering and pagination using **AJAX** (No page reloads).
* **Advanced Cart Management:** Real-time stock validation preventing users from adding out-of-stock items.
* **Order Tracking:** Dedicated "My Orders" section for users to track their purchase history and statuses.
* **Secure Authentication:** Robust Login/Register system using ASP.NET Core Identity.

### 🛡️ Admin & Backend Power
* **Role-Based Access Control (RBAC):** Strict separation between `Admin` and `User` roles.
* **Inventory Control:** Full CRUD operations for Products and Categories.
* **Security First:** Protected against SQL Injection and unauthorized routing access.
* **Data Integrity:** Solid database relationships and constraints using Entity Framework Core.

---

## 🛠️ Tech Stack & Architecture

* **Backend:** ASP.NET Core MVC, C#, LINQ
* **Architecture:** Repository Pattern, N-Tier Architecture concepts
* **Database:** SQL Server, Entity Framework Core (Code-First Approach)
* **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript, jQuery, AJAX
* **Identity & Security:** ASP.NET Core Identity
